Figura-model-Kandyyy25-

This is my model for Figura 0.1.6 (Minecraft Java 1.21.1). It includes 5 animated emotions and extra bones.

At the moment, I don't know much about programming in Lua, so this skin is simply my way of trying to learn by recreating things I find interesting and see other people doing. Because of that, it's noticeable that some animations are still not as smooth and natural as they could be.

I plan to polish not only the code but also the animations, as well as rename some things so they make more sense.

In the future, I want to add a system that displays injury textures based on the character's current health percentage, without interfering with the skin/outfit currently being used.

I also want to implement a system that keeps the selected skin from the "Outfits" section saved even after leaving and rejoining the game.

*The "Outfits" Action Wheel menu is not finished yet, since I have only created my main/default skin. This means that selecting any of the available options will cause the model to break, and it will need to be reloaded.*
